Introduction

The first thing that you must know is that Carrot is an experimental framework built as a learning project by an amateur programmer. It's not some super framework built by super coders (not that you should treat frameworks that way either). I guess what I'm trying to say is, expect some WTF moments, and bugs. Don't expect awesomeness.
