Welcome to Carrot

Carrot is an experimental framework that is created as a learning project. It quickly grows from a CodeIgniter-like framework into another beast of its own. It mandates the usage of dependency injection container and provider classes. Carrot is still very unstable, with many changes to the core planned.

Requirements

  • Willingness to work with dependency injection container and provider classes.
  • PHP 5.3+ (Carrot uses namespaces and anonymous functions).
  • Apache2 web server.
