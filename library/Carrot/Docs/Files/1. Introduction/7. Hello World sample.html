Hello World in Carrot

Please note that in Carrot, you can replace configuration classes with your own implementations, so this tutorial can only be guaranteed to work with default, out-of-the-box configurations.

Here are the list of steps you must do to create a simple hello world webpage in Carrot:

  • Create the HelloWorld class.
  • Configure dependency injections of HelloWorld.
  • Create and add a route that points to HelloWorld.

Of course in a real world application, you'll still have to think about layering and unit testing, but as an introduction to Carrot, this will do.
